# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/17gbkxq3ms7wSL4j4jalqWE-KCVEll_b9

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env` (use `.env.example` as a template) to your Gemini API key
3. Run the app:
   `npm run dev`

### Local Testing Build (Windows/Linux)
You can also use the provided scripts for a quick setup:
- **Windows**: Run `run.bat`
- **Linux**: Run `./run.sh`
